# SCADA-Dashboard-Simulation
This mini-project is a web-based Operational Technology (OT) monitoring dashboard designed to visualize offshore wind farm infrastructure. It integrates a custom Single Line Diagram (SLD) with real-time, simulated data to model voltage-based alarms, power generation trends, and interactive circuit breaker logic.
